How it works

From zero to live
in three steps

No complicated setup. No CLI. No config files. Just copy, edit, and deploy.

1

Copy the file

Download or clone the single HTML file. Everything — styles, scripts, markup — lives in one place. Open it in any editor.

2

Customise your content

Update the text, colours, sections and links to match your brand. The CSS variable system makes restyling a one-line change.

3

Push to Vercel

Drag the file into a new Vercel project or connect your GitHub repo. Vercel detects a static site and deploys instantly.
